Skenario: Pintu Belakang Gudang Digital (Directory Traversal)

Banyak sistem di dunia nyata mengalami kebocoran data karena admin lupa mengunci folder internal. Di sini Anda bertindak sebagai detektif yang memeriksa berkas cadangan rahasia yang tertinggal di area publik.

Level 1: Directory Traversal

Simulasi direktori sistem. Ketuk file untuk membuka.

📄readme.txt
📜system.log
🗄️database_backup_2026.conf
🗒️notes.old
🕶️.shadow
Butuh Bantuan?

Tier 1: Analogi

Bayangkan file manager yang tidak sengaja meninggalkan cadangan sensitif. Cari nama yang tampak bernilai tinggi.

Tier 2: Analisis

File cadangan konfigurasi sering berisi artefak rahasia. Sentuh untuk memicu pratinjau lokal.

Tier 3: Solusi Instan

database_backup_2026.conf

Skenario: Kasir Digital yang Lengah (Parameter Tampering)

Bayangkan sebuah toko ritel di mana pembeli bisa menulis sendiri harga barang di label sebelum dibawa ke kasir. Celah ini terjadi jika aplikasi memercayai input harga langsung dari HP pengguna tanpa verifikasi server pusat.

Level 2: Parameter Tampering

Rp
Butuh Bantuan?

Tier 1: Analogi

Seperti kasir yang hanya percaya tampilan angka tanpa validasi belakang layar.

Tier 2: Analisis

Jika harga bernilai 0 atau negatif, perhitungan bisa runtuh sehingga transaksi menjadi gratis.

Tier 3: Solusi Instan

0
-1

Skenario: Menipu Penjaga Gerbang Database (SQL Injection)

Sistem database membaca karakter khusus seperti tanda petik untuk menyusun perintah data. Peretas memanfaatkan kelemahan ini dengan menyelipkan karakter penipu untuk melewati validasi kata sandi tanpa tahu password aslinya.

Level 3: SQL Injection

Butuh Bantuan?

Tier 1: Analogi

Penjaga pintu bisa tertipu ketika kata sandi menyisipkan instruksi untuk mengabaikan pemeriksaan.

Tier 2: Analisis

Pola klasik mengandung ' OR ' atau komentar setelah pengguna admin.

Tier 3: Solusi Instan

admin' --
admin' OR '1'='1
x' OR '1'='1

Skenario: Papan Pengumuman Beracun (Cross-Site Scripting / XSS)

Celah ini terjadi ketika kotak ulasan umum dapat digunakan untuk menyuntikkan kode program aktif. Ketika orang lain membuka ulasan tersebut, browser mereka dipaksa menjalankan instruksi berbahaya secara otomatis.
